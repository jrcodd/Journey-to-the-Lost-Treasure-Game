Journey To The Lost Treasure

by: Jackson Codd

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I just finished a space-themed game and I wanted to create a interesting RPG game. I wanted to make a game with strategy rather then reflexes. I wanted to have fun with graphics and create a visually appealing game.

2. What I Learned

The user testers found some bugs in my code. They suggested that I make the game with instructions and to add a delay to the enemy attacks. They also suggested that I make the game slightly easier.

3. Architecture of My Game

My game has 19 classes each connected by the runner class, Game Panel, and Object Manager. The runner creates a game panel and object manager and those classes create the other individual classes.

4. Technical Implementation

My favorite aspect of the game is my enemy combat. It was hard to get the combat to work right. For example, the ship combat was difficult to make work properly because my code was not organized. To fix this problem, I organized my code and made more methods.

5. Most Challenging Part

The most challenging part of my game was keeping my code organized and neat. On some parts it was hard to read, so I had to make parts of it into methods. I added comments and organized my code to overcome this problem.

6. Future Fixes and Updates

If I were to improve my game, I would add a story and make more places to go with more characters. I would add armor and a more complex inventory system. I would add tamable creatures and more weapons.
